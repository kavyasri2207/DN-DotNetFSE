# Financial Forecasting

This folder shows a simple recursive forecasting example. The program starts with an initial value, applies a fixed growth rate, and repeats that calculation for the number of years you want to predict. The output below confirms the formula works and gives the final future value.

Run output:

```text
--- Financial Forecasting Test ---

Initial Value: $1000.00
Growth Rate: 5%
Time Periods: 10 years

Predicted Future Value: $1628.89

SUCCESS: Recursive forecasting algorithm executed perfectly.
```
