# E-commerce Platform Search Function

This folder contains a small Java example that compares linear search and binary search for an e-commerce product list. The run below shows both approaches finding the same product, and it highlights why binary search is a better fit when the data is sorted.

Run output:

```text
--- E-commerce Search Functionality Test ---

Starting Linear Search for Product ID: 107
Found via Linear Search: Product[ID=107, Name='Desk Lamp', Category='Furniture']

------------------------------------------------

Sorting array for Binary Search...
Starting Binary Search for Product ID: 107
Found via Binary Search: Product[ID=107, Name='Desk Lamp', Category='Furniture']
```
