# HTML5 Assessment - Community Events Website

## Project Overview
This is an HTML5 web application for a community events management system. The website allows residents to view upcoming events, browse past event galleries, and register for events.

## Features

### 1. Home Page
- **Welcome Section**: Greeting message for returning residents
- **Special Promotions**: Display of promotional offers (e.g., summer pool passes)
- **Announcements**: Important community notices (e.g., fee waivers)
- **Event Highlights**: Featured event promos with embedded media

### 2. Events Gallery
- **Past Events Gallery**: Showcase of highlights from recent city events
- **Double-Click to Enlarge**: Interactive feature to view images in detail
- **Image Grid**: Organized display of event photos in a responsive grid layout
- **Local Events Discovery**: "Find Local Events Near You" section with call-to-action button

### 3. Register Page
- **Event Registration Form**: User-friendly form to register for events
- **Form Fields**:
  - Full Name (text input)
  - Email Address (email input)
  - Phone Number (phone input)
  - Event Date (date picker)
  - Event Type (dropdown selector)
  - Message/Questions (textarea)
  - Character counter for messages
- **Clear Saved Preferences**: Button to reset form data
- **Submit Button**: "Complete Registration" action button

### 4. Help Center
- Dedicated support page for user assistance

## Technical Stack
- **HTML5**: Semantic markup and modern HTML features
- **CSS**: Styling and responsive design
- **JavaScript**: Interactive functionality and form handling

## Pages
- `index.html` - Home page with event promotions
- `help.html` - Help and support center
- Additional pages referenced in navigation:
  - Events Gallery
  - Register

## Navigation
The website includes a header navigation bar with links to:
- Home
- Events Gallery
- Register
- Help Center

## Styling
- Clean, modern design with a professional color scheme
- Responsive layout for desktop and mobile devices
- Light blue accent color for featured sections
- Green buttons for primary actions

## Key Components
- **Promotional Boxes**: Light blue background with important announcements
- **Image Gallery**: Responsive grid layout for event photos
- **Form Elements**: Accessible form controls with appropriate input types
- **Video Integration**: Embedded video player for event promos

## Usage
1. Open `index.html` in a web browser to view the home page
2. Navigate through the site using the header navigation menu
3. Fill out the registration form to register for events
4. Explore past events in the gallery section
