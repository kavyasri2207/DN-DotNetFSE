# ANLSSQl Assessment

This folder contains the completed ANSI SQL assessment for the Module 2 exercise set in `Module 2-ANSI SQL Using MySQL.pdf`.

## Files included
- `MySQL_Exercises.sql` — schema creation, sample dataset inserts, and SQL query answers for all assessment exercises.
- `README.md` — this file.

## What is included
- Table definitions for `Users`, `Events`, `Sessions`, `Registrations`, `Feedback`, and `Resources`.
- Sample insert statements matching the PDF exercise dataset.
- SQL query answers for all assessment requirements, including:
  - upcoming events per user
  - top-rated events
  - inactive users
  - peak session hours
  - city registration summaries
  - resource summaries
  - feedback alerts
  - event and organizer summaries
  - registration and session analytics
  - and more.

## How to use
1. Open MySQL Workbench or any MySQL client.
2. Create or select a test database.
3. Run the contents of `MySQL_Exercises.sql`.
4. Execute the query blocks in the file to verify each exercise result.

## Notes
- The SQL script is written for MySQL syntax.
- If your MySQL version requires a different date or enum handling, adjust the script accordingly.
- After running the script, use the query blocks in `MySQL_Exercises.sql` to confirm the exercise answers.
